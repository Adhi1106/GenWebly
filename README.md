# GenWebly

GenWebly is a simple AI-powered web app that helps generate website ideas and UI layouts from user prompts.  
It’s built to explore how AI can assist early-stage web design and prototyping.

## Live Demo
Try it out!: https://genwebly-v1.streamlit.app

## What this project does
- Takes a user prompt describing a website idea  
- Uses AI to generate UI and layout suggestions  
- Displays results through an interactive web interface  

This project focuses on combining **AI + web development** in a practical, beginner-friendly way.

## Tech Stack
- Python  
- Streamlit  
- Google Gemini API  

## Project Structure
```text
GenWebly/
├── app.py              # Main Streamlit application
├── requirements.txt    # Project dependencies
├── .gitignore          # Ignored files
├── .devcontainer/      # Development container (optional)
└── README.md
```


## How to run locally
1. Clone the repository
   
```text
git clone https://github.com/Adhi1106/GenWebly.git
```

2. Install dependencies
   
```text
pip install -r requirements.txt
```

3. Run the app
```text
streamlit run app.py
```
